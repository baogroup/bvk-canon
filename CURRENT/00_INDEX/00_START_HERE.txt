Назначение:
Этот файл является точкой входа в канон Балтийской Кооперативной Республики.

Главные правила:
- исходное создание архива и законов начинается с 2026 года
- 2086 год считается зрелым периодом Республики
- в CURRENT хранится только актуальная редакция
- GitHub History удерживает историю всех правок
- VERSIONS используется для отдельных крупных сохранённых редакций
- REPEALED используется для отменённых документов
- PUBLIC_READ показывает только документы уровня доступа PUBLIC
- assistant слой показывает документы уровней PUBLIC и SERVICE
- документы уровней RESTRICTED и SEALED не публикуются в открытый веб
- если PUBLIC-документ связан с закрытым слоем, это обязательно отмечается в шапке документа через поля закрытых связей и их коды
